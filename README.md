# 🇧🇩 Bangladesh 2.0 – Web Project

A modern **responsive landing page** that presents a vision for the future of Bangladesh through technology, sustainability, and digital transformation.

This project highlights different initiatives such as **innovation, development, digital infrastructure, and community engagement** through a clean and user-friendly interface.

---

## 🌐 Live Demo
🔗 **Live Website:**  
https://azizulislamch.github.io/Bangladesh-2.0-A1/

---

## 📖 Project Overview

**Bangladesh 2.0** represents a forward-looking vision where technology, sustainability, and inclusive growth shape the nation’s future.

The website showcases:

- Future development pillars
- Vision and innovation initiatives
- News and updates
- Donation support section
- Newsletter subscription

The goal of this project is to practice **modern HTML and CSS layout techniques** and build a **clean responsive landing page design**.

---

## 🚀 Features

✨ Modern landing page layout  
✨ Fully responsive design  
✨ Clean and structured UI  
✨ News section with articles  
✨ Donation section with predefined amounts  
✨ Newsletter subscription form  
✨ Hover effects and smooth UI interactions  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Google Fonts (Fira Sans)**
- **Flexbox Layout**
- **Responsive Design with Media Queries**
